
# Задание

Разработать класс String, для работы со строками. \
Класс должен содержать:
- Конструктор по умолчанию, позволяющий создать строку длиной 80 символов;
- Конструктор, позволяющий создавать строку произвольного размера;
- Конструктор, который создаёт строку и инициализирует её строкой, полученной от пользователя.
- Конструктопр копирования
- Конструктопр перемещения
- Необходимо создать деструктор, а также использовать механизмы делегирования конструкторов, если это возможно.
- создать статическую функцию-член, которая будет возвращать количество созданных объектов строк.  
- Перегрузите операторы + и += (сама конкатенация может быть с разными типами, подумайте об этом)
- Перегрузите операторы  << >> (ввод и вывод)
- Перегрузите операторы  < > == !=
- Перегрузите оператор [] — возвращает элемент по указанному индексу;
- Перегрузите оператор () — ищет символ в строке, если символ есть возвращает индекс, если нет –1.
- Преобразование к int возвращает длину строки.

Так же реализуйте все функции необходимые для коррктного использования вашего контейнера. Все, что посчитаете необходимым
