# Домашнее задание

# Задание.
Написать программу формирования частотного словаря появления отдельных слов в некоторомтексте. \ 
Обеспечить:
- Чтение исходного текста из файла;
- Вывод информации обо всех словах;
- Вывод информации о наиболее часто встречающемся слове;
- Запись результата в файл.
- 
При разработке программыиспользовать словарь типа map.


# Задание.

Группировка анаграмм \
Условие: \
Дан вектор строк {"cat", "dog", "tac", "god", "act"}. \
Сгруппируйте слова, являющиеся анаграммами (состоят из одних и тех же букв). \
Результат выведите в виде: каждая группа в отдельной строке (порядок групп не важен). \

Что можно попробовать использовать: std::map<std::string, std::vector<std::string>>, лямбду для сортировки букв в слове.

Пример вывода:
```
cat tac act
dog god
```

# Задание.
Сортировка студентов по нескольким критериям \
Условие: \
Определите структуру: \

```
struct Student {
    std::string name;
    int group;
    int grade; // оценка (чем выше, тем лучше)
};
```
Создайте вектор из 10 студентов. Отсортируйте их по группе (по возрастанию), а внутри одной группы — по убыванию оценки. 
Выведите отсортированный список в виде таблицы.
Что использовать: std::vector<Student>, std::sort с лямбдой, сравнивающей сначала group, затем grade.

```
Иванов, группа 101, оценка 85
Петров, группа 101, оценка 78
Сидоров, группа 102, оценка 92
```
