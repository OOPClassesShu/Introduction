# Задание.
Написать программу формирования частотного словаря появления отдельных слов в некоторомтексте. \ 
Обеспечить:
- Чтение исходного текста из файла;
- Вывод информации обо всех словах;
- Вывод информации о наиболее часто встречающемся слове;
- Запись результата в файл.
- 
При разработке программыиспользовать словарь типа map.


# Задание.

Группировка анаграмм \
Условие: \
Дан вектор строк {"cat", "dog", "tac", "god", "act"}. \
Сгруппируйте слова, являющиеся анаграммами (состоят из одних и тех же букв). \
Результат выведите в виде: каждая группа в отдельной строке (порядок групп не важен).

Что можно попробовать использовать: std::map<std::string, std::vector<std::string>>, лямбду для сортировки букв в слове.

Пример вывода:
```
cat tac act
dog god
```

# Задание.
Сортировка студентов по нескольким критериям \
Условие: \
Определите структуру:

```
struct Student {
    std::string name;
    int group;
    int grade; // оценка (чем выше, тем лучше)
};
```
Создайте вектор из 10 студентов. Отсортируйте их по группе (по возрастанию), а внутри одной группы — по убыванию оценки. 
Выведите отсортированный список в виде таблицы.
Что использовать: std::vector<Student>, std::sort с лямбдой, сравнивающей сначала group, затем grade.

```
Иванов, группа 101, оценка 85
Петров, группа 101, оценка 78
Сидоров, группа 102, оценка 92
```


# Задание. 

Гадание «Тайна трёх карт»

Вы — программист‑гадалка. \
Вам поступила колода игральных карт (36 штук: 6,7,8,9,10,Валет,Дама,Король,Туз четырёх мастей). \
Колода представлена в виде вектора строк, где каждая строка — карта, например: "6Ч", "KВ", "10Б", "AК". \
Пользователь вытягивает три случайные карты (без возвращения). Ваша задача — предсказать судьбу по правилам: \

- Если все три карты одной масти → «Большая удача!»
- Если есть пара одинакового достоинства (например, два короля) → «Неожиданный поворот».
- Если все карты чёрной масти (В К) → «Будьте осторожны».
- Если все карты красной масти (Ч Б) → «Любовь на горизонте».
- Иначе → «День пройдёт спокойно».

Но правила применяются по приоритету (сверху вниз): как только условие выполнилось, остальные не проверяются.

Технические требования
- Создайте вектор из всех 36 строк (карт). Перемешайте его с помощью std::shuffle.
- «Вытяните» первые три карты (первые три элемента после перемешивания).
- Для анализа используйте STL‑алгоритмы и лямбды:
- Разделите достоинство и масть (можно substr или поиск по последнему символу).
- Используйте std::all_of, std::count_if, std::adjacent_find и т.п.
- Выведите на экран три вытянутые карты и результат гадания.
