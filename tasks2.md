
## Задача. 

Система учёта сотрудников (с классом-пользователем)

Есть разные категории сотрудников:

- Обычный сотрудник (Employee) – базовый класс.
- Менеджер (Manager) – получает надбавку за размер команды.
- Разработчик (Developer) – получает надбавку за переработки.
- Стажёр (Intern) – фиксированная зарплата.

Класс "Компания" (Company) хранит список всех сотрудников и умеет:

- нанимать (добавлять) сотрудников,
- рассчитывать фонд оплаты труда за месяц,
- выводить детальную ведомость.

Технические требования

1. Базовый класс Employee \
Закрытые или защищённые поля:
- std::string name – имя сотрудника,
- int experience – стаж в годах.

Публичные методы:

- Конструктор с параметрами (имя, стаж).
- Метод double calculateSalary() const – для обычного сотрудника возвращает 30 000 + 1000 * стаж.
- Метод std::string getName() const.
- Деструктор.

2. Класс Manager – наследование от Employee \ 
Дополнительное поле: int teamSize (количество подчинённых).

Переопределённый метод calculateSalary(): \
базовая_зарплата(стаж) + 5000 * teamSize.

Конструктор, инициализирующий базовую часть и teamSize.

3. Класс Developer – наследование от Employee
Дополнительное поле: int extraHours (сверхурочные часы за месяц).

Переопределённый метод calculateSalary():
базовая_зарплата(стаж) + 1000 * extraHours.

Конструктор.

4. Класс Intern – наследование от Employee
Стаж у стажёра всегда 0 (можно передавать 0 в конструктор базы или игнорировать стаж).

Переопределённый метод calculateSalary(): возвращает 15 000 (без учёта стажа).

5. Класс-пользователь Company (управляет сотрудниками) \ 
Поля:

- std::vector<Employee*> employees – хранит указатели на базовый класс (полиморфный контейнер).
- (Опционально) std::string companyName.

Публичные методы:

- void hire(Employee* emp) – добавляет сотрудника в список (принимает указатель на Employee, что позволяет передавать new Manager(...), new Developer(...) и т.д.).
- double calculateTotalSalary() const – проходит по всем сотрудникам, суммирует calculateSalary().
- void printPayroll() const – выводит таблицу: имя, должность (определить отдельным методом getRole() ), зарплата.
- ~Company() – освобождает память (удаляет всех сотрудников через delete).

Пример структуры вывода printPayroll():
```
=== Payroll for Company "SoftLab" ===
Alice (Manager)       : 45000 rub
Bob (Developer)       : 38000 rub
Charlie (Intern)      : 15000 rub
Total salary fund     : 98000 rub
```
